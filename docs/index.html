What is AutoUAT

AutoUAT is a stochastic UAT/UX testing engine. Point it at any URL and it automatically discovers pages, generates test scenarios, executes them in real browsers, and produces structured reports with prioritized issues.

Unlike traditional E2E tools, AutoUAT does not require you to write test scripts. It automatically focuses on the parts of your app most likely to have bugs - no test scripts required.

Core capabilities

Discovery
Crawls your site, finds routes, forms, auth walls, and interactive elements automatically.
Scenario generation
Creates role/viewport/flow combinations from discovered structure using adaptive sampling.
Browser execution
Runs scenarios in real Chromium, Firefox, WebKit, or Electron via Playwright with screenshot and DOM capture at every step.
Heuristic evaluation
11 production-ready evaluators score each scenario for layout, accessibility, content, performance, and more.
LLM critic panel
Optional multi-agent LLM panel reviews findings and reaches consensus on real issues.
Triage + dedup
Automatic severity scoring deduplicates findings and ranks issues by priority.

Three interfaces

Desktop app
Electron GUI with visual run setup, live progress, settings, and run history.
